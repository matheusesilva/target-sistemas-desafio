# DESAFIO

Este repositório contém 3 programas independentes em C# para resolver o desafio para a vaga de Desenvolvedor de Sistemas Jr.

## Programas
### 1. CalcularComissao - Cálculo de Comissões de Vendas

#### Desafio: Calcular comissões de vendedores baseado em faixas de valor

  **Regras:**

- Vendas < R$ 100,00: 0%
- Vendas < R$ 500,00: 1%
- Vendas ≥ R$ 500,00: 5%

**Entrada:** vendas.json

**Saída:** comissoes.json com totais por vendedor

### 2. ControleEstoque - Sistema de Controle de Estoque

#### Desafio: Gerenciar entradas e saídas de produtos

  **Funcionalidades:**

 - Movimentações com ID único
 - Descrição da movimentação realizada

**Entrada:** estoque.json

**Saída:** Estoque atualizado + movimentacoes.json

### 3. CalcularJurosAtual - Calculadora de Juros

#### Desafio: Calcular juros por atraso com multa diária
  **Regras:**
  
 - Taxa: 2,5% ao dia

**Entrada:** Valor + data de vencimento

**Saída:** Valor total com juros no console
